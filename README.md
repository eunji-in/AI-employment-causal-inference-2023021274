# AI and Youth Employment: Causal Inference Analysis

## Overview
Final project for Causal Inference and AI.

Research Question:
"Does the advancement of generative AI causally increase youth employment anxiety?"

## Repository Structure

- articles/ : collected news articles
- cleaned/ : cleaned text corpus
- extractions/ : entities and causal assertions
- knowledge_graph/ : knowledge graph files
- causal_dag/ : DAG and causal analysis
- report/ : final report

## Methodology

1. Corpus Collection
2. Text Cleaning
3. Information Extraction
4. Knowledge Graph Construction
5. Causal DAG Design
6. Causal Inference Analysis
